# Simvado Documentation

**Simvado** is a Decision-Intelligence Simulation Platform for professional training.

**Vision:** Become the global platform for professional simulation-based learning — the Netflix of Training Simulations.

**Founder:** Julius Joaquin

---

## Purpose of This Documentation

1. **Internal clarity** — single source of truth for all product, architecture, and business decisions
2. **AI-agent context** — structured reference for Claude, GPT, and automation agents
3. **Development guide** — maps directly to the codebase, schemas, and API contracts
4. **Investor communication** — publishable to Notion or pitch decks

---

## Flagship Product

**Boardroom Under Pressure** — an interactive cinematic simulation for corporate boards and executive leadership teams.

## Canonical Phase Definitions

All documents reference these phases consistently:

| Phase | Name | Scope |
|-------|------|-------|
| **Phase 1** | Foundation | Launch Boardroom Under Pressure. Individual + pilot enterprise access. Internal studio only. |
| **Phase 2** | Enterprise Scale | Enterprise dashboard, seat management, analytics. Add 3 more executive simulations. |
| **Phase 3** | Marketplace | Open platform to third-party simulation studios. Revenue-share model. |
| **Phase 4** | Global Expansion | Multi-language, multi-industry. Full simulation library across verticals. |
